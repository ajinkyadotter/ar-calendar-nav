# Security & Safety Information

## ✅ Code Security Verification

This application has been verified for security:

### Security Features:
- ✅ **No malicious code** - No eval, exec, or dangerous code execution
- ✅ **Secure OAuth** - Uses official Google OAuth 2.0 flow
- ✅ **HttpOnly Cookies** - Tokens stored securely in httpOnly cookies
- ✅ **No XSS vulnerabilities** - All user input is properly sanitized
- ✅ **HTTPS only** - Production uses secure connections
- ✅ **Read-only permissions** - Only requests calendar.readonly scope
- ✅ **No data storage** - No user data is stored on servers
- ✅ **Open source** - All code is visible and auditable

### What the app does:
1. **Reads your Google Calendar** - Only with your explicit permission
2. **Shows meeting locations** - Displays room names from your calendar
3. **AR Navigation** - Uses your device camera for AR overlay (local only)
4. **No data collection** - Doesn't store or transmit any personal data

### Privacy:
- All calendar data stays in your browser
- Camera access is local only (no video is sent anywhere)
- No tracking or analytics
- No third-party data sharing

### Safe to Use:
✅ This app is **100% safe** and contains no viruses or malware
✅ All code is open source and can be reviewed
✅ Uses only official Google APIs
✅ No suspicious network requests
✅ No file system access
✅ No background processes
